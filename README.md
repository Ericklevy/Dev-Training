## Installation

**Languages**: Python<br>

## Uso 

**Python must be installed to use the dependencies**

### How to Execute

```bash
    pip install -r requirements.tx
```
```bash
    cd/src
```
```bash
    python spreadsheet.py
```

### How to Use with Python Virtual Environment

```bash
    python -m venv venv
```
```bash
    source venv/bin/activate
```
```bash
    pip install -r requirements.txt
```
```bash
    cd/src
```
```bash
    python spreadsheet.py
```
